title: "Customizing Meeting Types"
---

<Accordion title="Customizing Prompts at Meeting Level">
  1. Head to any meeting that is ‘Ready for Review’
  2. At the top of the meeting notes, click the button that says ‘Customize Prompt’
  3. Edit the prompt
  4. Click ‘Save and Regenerate’

  ![Trainhazelmeetingtype](/images/trainhazelmeetingtype.gif)
</Accordion>

<Accordion title="Customizing Prompts within Meeting Settings">
  1. Head to Settings / Meeting Settings
  2. You can either add a new Meeting Type or edit existing Meeting Types.

  ![Addmeetingtype](/images/addmeetingtype.gif)

  4. Select the Meeting Type you would like to edit, or click ‘Add Meeting Type’ to create a new meeting type.
  4. Click ‘Save and Regenerate’

  <Tip>
    If you don’t want to start from scratch, you can duplicate the ‘General Meeting’ type prompts by creating a new meeting type and only filling in the Name, then clicking save.

    ![Duplicatemeetingtype](/images/duplicatemeetingtype.gif)
  </Tip>
</Accordion>
